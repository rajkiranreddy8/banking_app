# 💸 The People's Bank

The People's Bank is a simple banking application that allows users to:

✅ Create Users  
✅ View the List of Users  
✅ Transfer Money Between Users  
✅ View Transaction History  
✅ Delete Users  

---

## 📚 Features

- **Create User:** Add a new user with initial balance.
- **User List:** View all registered users.
- **Transfer Money:** Transfer funds between users with balance validation.
- **Transaction History:** View all completed transactions.
- **Delete User:** Delete a user and their data.

---

## ⚙️ Tech Stack

- **Frontend:**
  - React JS
  - React-Bootstrap
  - Axios for API requests
- **Backend:**
  - Node.js
  - Express.js
  - MongoDB (Mongoose for ORM)
